Treeview

Treeview is a visual portrayal of hierarchical data that starts with the root item and progresses through its children and their descendants. Aside from the root, every item has a parent and can have children. To create this Treeview, we used Bootstraps Collapse feature.

Basic Example

Use data-show="true" attribute with the collapse element to expand the tree at first.
